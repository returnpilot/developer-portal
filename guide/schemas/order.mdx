---
title: 'Order'
---

<ResponseField name="id" type="integer" required>
    ID of the order.
</ResponseField>

<ResponseField name="origin_address" type="array" required>
    The [Address](/guide/schemas/address) representing where the return originates.
</ResponseField>

<ResponseField name="destination_address" type="array" required>
    The [Address](/guide/schemas/address) representing where the return is shipped.
</ResponseField>

<ResponseField name="parcels" type="array" required>
    An `array` of [Parcels](/guide/schemas/parcel) associated with the order.
</ResponseField>

<ResponseField name="reference" type="string" required>
    The unique number or reference code for the order.
</ResponseField>

<ResponseField name="status" type="integer" required>
    A [Status](/guide/enums/shipment-statuses) enum representing the status of the order's return shipment.
</ResponseField>

<ResponseField name="price" type="decimal">
    The total cost for the return shipment in your organization's currency. If the order does not have a return shipment, then it will be `null`.
</ResponseField>

<ResponseField name="carrier" type="string">
    The name of the provider handling the return shipment. If the order does not have a return shipment, then it will be `null`.
</ResponseField>

<ResponseField name="return_url" type="string" required>
    The self-expiring link used to schedule a return.
</ResponseField>

<ResponseField name="expires_at" type="string" required>
    Timestamp of when a return can no longer be scheduled.
</ResponseField>

<ResponseField name="created_at" type="string" required>
    Timestamp of when the order was created.
</ResponseField>

<ResponseField name="updated_at" type="string" required>
    Timestamp of when the order was last updated.
</ResponseField>
