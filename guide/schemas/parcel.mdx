---
title: 'Parcel'
---

<ResponseField name="id" type="integer" required>
    ID of the parcel.
</ResponseField>

<ResponseField name="product" type="array" required>
    The [Product](/guide/schemas/product) details of the parcel.
</ResponseField>

<ResponseField name="status" type="integer" required>
    A [Status](/guide/enums/parcel-statuses) enum representing the status of the parcel's return shipment.
</ResponseField>

<ResponseField name="returnable" type="boolean" required>
    Status indicating whether the parcel may be returned.
</ResponseField>

<ResponseField name="reason" type="integer">
    A [Reason](/guide/enums/reason-statuses) enum representing the reason the parcel was returned. If the parcel has not been returned, then it will be `null`.
</ResponseField>

<ResponseField name="tracking_number" type="string">
    The tracking number for the parcel. If the parcel has not been returned, then it will be `null`.
</ResponseField>

<ResponseField name="tracking_url" type="string">
    The URL to the tracking page for the parcel. If the parcel has not been returned, then it will be `null`.
</ResponseField>

<ResponseField name="shipping_label" type="string">
    The URL to the shipping label to be affixed to the parcel. If the parcel has not been returned, then it will be `null`.
</ResponseField>

<ResponseField name="created_at" type="string" required>
    Timestamp of when the parcel was created.
</ResponseField>

<ResponseField name="updated_at" type="string" required>
    Timestamp of when the parcel was last updated.
</ResponseField>
