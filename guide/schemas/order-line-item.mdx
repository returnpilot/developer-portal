---
title: 'Order Line Item'
---

<ResponseField name="id" type="integer" required>
    ID of the order line item.
</ResponseField>

<ResponseField name="product" type="array" required>
    The [Product](/guide/schemas/product) details of the order line item.
</ResponseField>

<ResponseField name="reason" type="integer">
    A [Reason](/guide/enums/reason-statuses) enum representing the reason the order line item was returned. If the order line item has not been returned, then it will be `null`.
</ResponseField>

<ResponseField name="created_at" type="string" required>
    Timestamp of when the order line item was created.
</ResponseField>

<ResponseField name="updated_at" type="string" required>
    Timestamp of when the order line item was last updated.
</ResponseField>
