### Overview

The ReturnPilot API returns responses in JSON format and uses HTTP status codes to indicate the success or failure of a request.

For example, the response a successful request to retrieve an address will have a `200` status code and the following body:

```json
{
    "data": {
        "id": 1,
        "name": "Kohler Group",
        "email": "art.port.example@example.com",
        "phone": "+123456789",
        "line_1": "123 Example Street",
        "line_2": "Apt. 300",
        "city": "Moenburgh",
        "region": "Colorado",
        "post_code": "72099",
        "country": 1,
        "created_at": "2024-12-09 13:49:57",
        "updated_at": "2024-12-09 13:49:57"
    }
}
```
