### Overview

This developer portal contains the documentation required to assist software developers in working with the ReturnPilot API. Follow the steps below to get started building your integration.

<Steps>
    <Step title="Get an API token">
        Begin by visiting your organization's ReturnPilot account and [creating an API token](https://app.returnpilot.co/organization/integrations) to authenticate with.
    </Step>
    <Step title="Learn how to authenticate">
        Discover how to [communicate with the ReturnPilot API](/guide/concepts/requests) by embedding your API token within your requests.
    </Step>
    <Step title="Create your application">
        Explore each of the available API endpoints, then design and develop your integration to work with them.
    </Step>
</Steps>

If you encounter a bug with the API, then please create an issue on the relevant GitHub repository, or contact us via the support center.
