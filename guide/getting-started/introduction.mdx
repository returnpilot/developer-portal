### Overview

This developer portal contains the documentation required to assist software developers in working with the ReturnPilot API. Follow the steps below to get started building your integration.

<Steps>
    <Step title="Get an API token">
        Begin by visiting your organization's ReturnPilot account and [creating an API token](https://app.returnpilot.co/organization/integrations) to authenticate with.
    </Step>
    <Step title="Learn how to authenticate">
        If you are not using one of our SDKs, then your next step is to discover how to [authenticate with our API](/guide/concepts/requests).
    </Step>
    <Step title="Build your integration">
        Check if we have an [SDK for your programming language](/guide/getting-started/libraries), or build your own integration by exploring the API reference.
    </Step>
</Steps>

If you encounter a bug with the API or with one of our SDKs, then please create an issue on the relevant GitHub repository, or contact us via the ReturnPilot  support center.
