### Overview

This endpoint can be used to retrieve an order.

Endpoint: `/api/v1/orders/{order_id}`

Method: `GET`

### URL Parameters

<OrderIdField action="retrieve"/>

### Response

If the order is successfully retrieved, the API will return a `200 OK` status code.

The response will contain the following content:

<ResponseField name="data" type="array" required>
    An array containing a single [Order](/guide/schemas/order) object.
</ResponseField>
