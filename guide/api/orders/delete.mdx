### Overview

This endpoint can be used to delete an order.

Endpoint: `/api/v1/orders/{order_id}`

Method: `DELETE`

<Note>
    Orders that have returns (completed or otherwise) cannot be deleted.
</Note>

### URL Parameters

<OrderIdField action="delete"/>

### Response

If the order is successfully deleted, the API will return a `204 No Content` status code.

The response will not contain any content.

<ResponseExample>
    ```text 204
    No Content
    ```
</ResponseExample>
