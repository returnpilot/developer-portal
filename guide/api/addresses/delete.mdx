### Overview

This endpoint can be used to delete an address.

Endpoint: `/api/v1/addresses/{address_id}`

Method: `DELETE`

### URL Parameters

<AddressIdField action="delete"/>

### Response

If the address is successfully deleted, the API will return a `204 No Content` status code.

<ResponseExample>
    ```text 204
    No Content
    ```
</ResponseExample>
