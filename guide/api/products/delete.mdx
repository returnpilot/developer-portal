### Overview

This endpoint can be used to delete a product.

Endpoint: `/api/v1/products/{product_id}`

Method: `DELETE`

### URL Parameters

<ProductIdField action="delete"/>

### Response

If the product is successfully deleted, the API will return a `204 No Content` status code.

The response will not contain any content.

<ResponseExample>
    ```text 204
    No Content
    ```
</ResponseExample>
