### Overview

This endpoint can be used to retrieve products.

Endpoint: `/api/v1/products/`

Method: `GET`

<PaginationNotice />

### Response

If the products are successfully retrieved, the API will return a `200 OK` status code.

The response will contain the following content:

<ResponseField name="data" type="array" required>
    An `array` of [Product](/guide/schemas/product) objects.

    If no products are found, this will be an empty `array`.
</ResponseField>
