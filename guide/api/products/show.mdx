### Overview

This endpoint can be used to retrieve a product.

Endpoint: `/api/v1/products/{product_id}`

Method: `GET`

### URL Parameters

<ProductIdField action="retrieve"/>

### Response

If the product is successfully retrieved, the API will return a `200 OK` status code.

The response will contain the following content:

<ResponseField name="data" type="array" required>
    An array containing a single [Product](/guide/schemas/product) object.
</ResponseField>
