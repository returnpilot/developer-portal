### Overview

This endpoint can be used to update a product.

Endpoint: `/api/v1/products/{product_id}`

Method: `PATCH`

### URL Parameters

<ProductIdField action="update"/>

### Request

<ResponseField name="name" type="string" required>
    The name of the product. For example, "Red Shirt" or "SKU 1356284".

    Minimum character length: `2`

    Maximum character length: `50`
</ResponseField>

<ResponseField name="category" type="string" required>
    The category of the product. For example, "Clothing".

    Minimum character length: `2`

    Maximum character length: `50`
</ResponseField>

<ResponseField name="value" type="decimal" required>
    The value of the product in USD. For example, 55.21.

    Minimum: `1.00`

    Maximum: `300.00`
</ResponseField>

<ResponseField name="length" type="integer" required>
    The length of the product in inches (rounded up).

    Minimum: `4`

    Maximum: `25`
</ResponseField>

<ResponseField name="width" type="integer" required>
    The width of the product in inches (rounded up).

    Minimum: `4`

    Maximum: `25`
</ResponseField>

<ResponseField name="height" type="integer" required>
    The height of the product in inches (rounded up).

    Minimum: `1`

    Maximum: `25`
</ResponseField>

<ResponseField name="weight" type="integer" required>
    The height of the product in pounds (rounded up).

    Minimum: `1`

    Maximum: `25`
</ResponseField>

### Response

If the product is successfully updated, the API will return a `204 No Content` status code.
