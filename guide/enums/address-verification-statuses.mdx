### Overview

The following enumerated values represent each of the address verification statuses that are supported by the ReturnPilot API:

| ID  | Status          | Description                                         |
|-----|-----------------|-----------------------------------------------------|
| 900 | Accepted        | The address has been verified.                      |
| 901 | Invalid         | A matching address could not be found.              |
| 902 | Recommended     | Address not found, but a similar one exists.        |
| 903 | Not Recommended | A similar address exists, but its accuracy debated. |
