### Overview

The following enumerated values represent each of the box types that are supported by the ReturnPilot API:

| ID | Status  | Description                                    |
|----|---------|------------------------------------------------|
| 1  | Small   | A small box for a limited number of items      |
| 2  | Medium  | A medium box for a greater number of items     |
| 3  | Large   | A large box for a considerable number of items |
